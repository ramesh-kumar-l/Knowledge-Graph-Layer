# 00 — Project Vision

## What we are building
The **Knowledge Graph Layer** for SCP (Semantic Control Plane). It transforms raw,
disconnected memories from the SCP Memory Core into structured, connected, explainable
semantic knowledge.

## The transformation
```
Raw Data → Information → Knowledge → Understanding → Action
```
The layer converts disconnected memories into connected knowledge where every fact is:
- Explainable
- Traceable
- Auditable
- Trustworthy
- Versioned

## What it is NOT
- Not merely a graph database.
- Not a replacement for any existing SCP foundation.

## What it enables
Persistent Memory · Knowledge Discovery · Explainable Retrieval · Goal Tracking ·
Relationship Intelligence · Semantic Reasoning · Trustworthy AI Systems.

## North star
Become the reliable semantic memory infrastructure and foundational component of SCP —
usable by engineers, AI agents, enterprises, and future AI-native operating systems.
Every decision must still make sense five years from now.
