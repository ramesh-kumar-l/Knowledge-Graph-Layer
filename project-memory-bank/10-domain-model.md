# 10 — Domain Model

**Status:** Not started — Phase 1 deliverable.

## Purpose
Define the conceptual domain of the Knowledge Graph Layer: the core objects (Entity,
Relationship, Evidence, Provenance), their fields and invariants, and how memories map
into them. Language-neutral (see DEC-0001).

## To be defined in Phase 1
- Core object definitions and required fields.
- Entity vs. relationship boundaries.
- How Memory Core records become graph objects.
- Versioning and soft-delete semantics at the model level.
